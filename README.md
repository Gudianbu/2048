# 2048 Game
这是用C#写的一个WindowsForms应用程序.
是一个传统的版本，并无特殊之处，代码中含有详细的注释。

# VS版本
用vs2013及以上版本打开，推荐使用vs2015或者2017.

# 操作方式
1. 方向键移动方块<br />
2. shife+方向键上下（↑↓）调整窗体透明度<br />
3. F1打开帮助<br />
4. F2保持窗体在最前<br />
5. F3锁定/解锁窗体<br />
6. F4隐藏任务栏图标<br />
7. F5重新开始<br />
8. F6保存截图，截图在应用程序根目录<br />
9. ESC退出游戏<br />

* 游戏会自动保存
